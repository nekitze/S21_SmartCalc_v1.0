Smart Calc v1.0

About

  • This program evaluates arithmetic expressions in infix notation. Also Smart Calc can build graphs and calculate the amount of payments for loans.

How to install:

1) make install
2) Launch Smart Calc from Desktop icon

How to use:

1) Use it like basic calculator.
2) Button Graph to open graph plotter window.
3) Button Credit to open payments calculator.
4) Better sse brackets for unary operations (-a) (+b)

Requirements:

1) QT
2) qmake
3) check.h & lcov for tests
4) gcc
